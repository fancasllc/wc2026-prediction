# PROJECT_OVERVIEW.md

## App Purpose

`wc2026-prediction` は、2026年ワールドカップを題材にした無料の予想投票サイトです。

ユーザーは実際のお金を賭けず、架空ポイントを使って試合やテーマに投票します。結果確定後、的中者へ総プールに応じたポイントが還元され、個人別の確定収支やランキングで盛り上がりを作ります。

## Target Users

- サッカーに詳しい参加者
- サッカーに詳しくないが、イベントとして楽しみたい参加者
- 友人、社内、コミュニティなどでワールドカップ予想を楽しむユーザー
- 管理者として試合登録、結果確定、バックアップ確認を行う運営者

## Problems This App Solves

- ワールドカップ期間中に、参加者が気軽に予想で盛り上がれる場を作る
- お金を賭けずにスポーツベッティング風のゲーム性を提供する
- 個人別の投票状況、収支、ランキングを見える化する
- 管理者が試合登録、結果確定、データ確認を行いやすくする
- 万が一に備え、投票データをバックアップできるようにする

## Core Workflows

### User Workflow

1. 受付中の予想テーマを見る。
2. 試合詳細で選択肢、オッズ、参考情報、投票状況を見る。
3. 名前、投票ポイント、選択肢を入力する。
4. 投票確認画面で、投票後のオッズ変動と想定収支を確認する。
5. 投票する。
6. 投票状況、個人別収支、ランキングを見る。
7. 投票後5分以内、締切前、かつ最新投票であれば削除できる。

### Admin Workflow

1. 管理者パスワードで認証する。
2. 試合を個別登録または CSV 登録する。
3. 必要に応じて試合情報を編集する。
4. 締切済みの試合結果を入力し、確認後に確定する。
5. 投票 DB、ユーザー DB、試合 DB、バックアップ状況を確認する。
6. 手動バックアップを作成する。

### Backup Workflow

1. 定期的に DB 情報を CSV 化する。
2. Cloudflare R2 に外部保存する。
3. 管理画面からバックアップ履歴と CSV を確認する。
4. 試合結果確定後にもバックアップを作成する。

## Accidents To Avoid

- 本番 DB の誤削除、初期化、上書き
- 投票済みデータと選択肢の不整合
- 結果確定後の投票削除や再計算ミス
- ハンデ設定と結果判定のズレ
- 参考オッズや外部リンクの誤表示
- `wc2026-prediction` と `wc2026-member-prediction` の取り違え
- 管理者パスワードや API キーなどのシークレット流出
- バックアップ未作成のまま重要な操作を進めること
